# **App Name**: CozyColor

## Core Features:

- Doodle Display: Display a collection of cozy doodle illustrations for users to view and select for coloring.
- Digital Coloring: Allow users to digitally color the doodles with a variety of colors and brush sizes directly within the app.
- Color Palette Suggestion: AI tool suggests color palettes based on the selected doodle to enhance the coloring experience.
- Downloadable PDF: Allow users to download the colored or uncolored doodles as a PDF for printing and physical coloring.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C2) to create a warm and inviting atmosphere, reflecting the 'cozy' theme.
- Background color: Pale pink (#FDF2F4) to maintain a light and airy feel, complementing the primary color.
- Accent color: Light orange (#F4C7B4) to add a touch of warmth and contrast, highlighting key interactive elements.
- Font: 'Poppins', a geometric sans-serif, for a contemporary and readable user interface.
- Use simple, hand-drawn style icons to align with the cozy and illustrative theme.
- Clean and minimalist layout to ensure ease of navigation and focus on the doodle illustrations.
- Subtle and gentle animations for transitions and interactions, enhancing the feeling of relaxation.
